## The Easiest Way (Literally 2 minutes):

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/gunfight-arena.git
git branch -M main
git push -u origin main
```

### 2. Deploy to Railway
1. Go to **[railway.app](https://railway.app)**
2. Click **"Start a New Project"**
3. Sign up with GitHub
4. Click **"Deploy from GitHub"**
5. Select **YOUR repository**
6. Click **"Deploy Now"**
7. Wait ~30 seconds
8. Get the generated URL

### 3. Share & Play!
- Copy the URL (looks like `https://gunfight-arena-production.up.railway.app`)
- Send to friends
- They click the link and play!
