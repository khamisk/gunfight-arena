## Super Easy Steps (5 minutes):

### Step 1: Create Railway Account
1. Go to [railway.app](https://railway.app)
2. Click **"Start a New Project"**
3. Sign up with GitHub (or email)

### Step 2: Deploy from GitHub
1. Click **"Deploy from GitHub"**
2. Click **"Configure GitHub App"** if needed
3. Select this repository
4. Click **"Deploy"**

Railway will automatically:
- Install dependencies
- Start the server
- Give you a live URL

### Step 3: Share the URL
Railway generates a URL like: `https://gunfight-arena-production.up.railway.app`

**That's it!** Share this URL with your friends. They can open it in their browser and play!

## If You Don't Have GitHub:

### Option: Use Replit (Even Easier)

1. Go to [replit.com](https://replit.com)
2. Click **"+ Create Repl"**
3. Choose **"Node.js"**
4. Upload files from your `1v1game` folder
5. Click **"Run"**
6. Copy the URL from the popup

## Troubleshooting:

**"Port not available"** - Don't worry, hosting sites handle this automatically

**"Connection refused"** - Wait 30 seconds for server to start after deployment

**Crashes** - Check the logs in Railway dashboard for errors
